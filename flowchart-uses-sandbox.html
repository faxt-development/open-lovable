Legend

Frontend (Blue): React-based UI components that users interact with.

API Endpoints (Green): Serverless functions that handle specific operations.

External Services (Yellow): Third-party services and tools the application integrates with.

Key Features:

  • AI-assisted code generation and editing
  • Live preview with Vite dev server
  • File system management
  • Web scraping capabilities
  • Sandboxed code execution
  • Real-time error monitoring
