Legend

Frontend (Blue): React-based UI components that users interact with.

API Endpoints (Green): Serverless functions that handle specific operations.

External Services (Yellow): Third-party services the application integrates with.

Local Development (Purple): Components that interact with the local file system and development server.

Key Features:

  • AI-assisted code generation and editing
  • Live preview with local development server
  • Local file system management
  • Code analysis and dependency detection
  • Real-time error monitoring
  • Project download and sharing
